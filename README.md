A German / English wordclock for the Pixoo64 from Divoo !

Call this script from an system inside your network to send the bitmap to your pixoo64

Use --once to call it one time, then exit !

---

python3 pixoo64_wortuhr.py 

usage: wortuhr.py [-h] --ip IP [--port PORT] [--fg R G B] [--dim R G B] [--bg R G B] [--dot R G B]
                  [--brightness BRIGHTNESS] [--interval INTERVAL] [--once]

                  
wortuhr.py: error: the following arguments are required: --ip
